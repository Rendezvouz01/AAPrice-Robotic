# **App Name**: AAPrice Robotics Landing Page

## Core Features:

- Header with Navigation: Sticky header with AAPrice Robotic Engineering LLC logo and navigation links.
- Hero Section: Display a hero section with a headline and high-tech robotic background.
- Product Grid: Showcase product categories in a responsive 3-column grid layout.
- Footer with Headquarters Info: Display the company's headquarters locations (Toronto, St. Louis) in the footer.
- Responsive Layout: Ensure all components adapt to different screen sizes using a mobile-first approach.

## Style Guidelines:

- Primary color: Dark navy blue (#2C3E50) to convey a sense of professionalism and reliability, aligning with the industrial sector.
- Background color: Light industrial gray (#F0F4F8), providing a clean, modern backdrop that allows the primary content to stand out. 
- Accent color: A brighter, analogous blue (#3498DB) used sparingly for call-to-actions and key highlights.
- Headline font: 'Space Grotesk', a sans-serif font for headlines, providing a computerized, techy, scientific feel, consistent with the robotics theme.
- Body font: 'Inter', a sans-serif font for body text, ensuring readability and a clean, modern appearance. 
- Lucide-Vue-Next icons: Use modern, geometric icons from Lucide-Vue-Next to represent product features and benefits, adding a visual element without overwhelming the design.
- Grid system: Implement a flexible grid system for product display and content arrangement to ensure a responsive layout across all devices.
- Subtle transitions: Incorporate subtle transitions and animations on hover states and content updates to enhance user engagement and provide visual feedback.
